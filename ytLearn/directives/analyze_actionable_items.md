# Analyze Actionable Items Directive

## Goal
Deeply analyze video transcripts to extract suggestions, advice, and practices. The output must be understandable without watching the video and machine-readable for downstream AI agents.

## Inputs
- **Video URL** or **Playlist URL**
- **Transcript JSON**: Generated by `get_transcript.py`.

## Tools
- `execution/analyze_transcript.py`: The specialized AI analysis script.

## Output
- `transcripts/{Video_Title}_analysis.json`: Structured data for AI agents.
- `transcripts/{Video_Title}_analysis.md`: Human-readable report.

## Procedure
1.  **Fetch Transcript**: Ensure the transcript JSON exists (run `get_transcript.py` if needed).
2.  **Run Analysis**:
    ```bash
    python execution/analyze_transcript.py --json_path "transcripts/Video_Title.json" --output_prefix "transcripts/Video_Title"
    ```
3.  **Review**: Check the generated `.md` file for clarity and the `.json` file for structure.

## Edge Cases
- **Long Transcripts**: The script should handle large contexts (Gemini Flash has 1M token window).
- **No Advice Found**: The AI should return an empty list rather than hallucinating.
